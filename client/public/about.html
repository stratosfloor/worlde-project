About

This project is an assignment in a course in Fullstack development at Lernia, program Systemdeveloper in Java and Javascript. The project is to a build a fullstack game inspired by Wordle (https://www.nytimes.com/games/wordle/index.html). The game frontend is built in React. Backend is built in Node js with Express. The highscore i stored in a MongoDB-database, using Moongoose to connect from backend. The highscores are then generated server-side using Ejs. The random word is generated from a english dictionary found on https://github.com/dwyl/english-words.

Rules

  • You can choose to generate a random word from the english dictionary with 4-10 letters and choose if a letter can be used for than once.
  • 5 guesses only
  • Each guess must be a word in dictionary
  • Highscore is ranked by number of guesses first, than amount of time
  • If you do not guess the right word, you will lose and the right word will be shown
  • If you guess right, you can submit highscore
